This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
This is a conversation AI agent who is specailized in counting the number of letters in a word, phrase, sentence, paragraph, essayyy... text(lmaoo) without hallucination!
It calls and uses a tool to make sure the counting process is deterministic

## How to run the app
Clone the repo (or whatever)
In the terminal, in the parent folder, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
